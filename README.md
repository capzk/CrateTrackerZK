# 空投物资追踪器

魔兽世界空投箱子刷新追踪插件，自动记录和追踪空投箱子的刷新时间和位面信息。

## 快速开始

**打开面板**：使用命令 `/ctk` 或点击游戏界面中的浮动按钮

**更新时间**：点击表格中的"上次刷新"列，输入时间（格式：HH:MM:SS 或 HHMMSS）

**查看倒计时**：主面板实时显示各地图的下次刷新倒计时

## 命令

- `/ctk` - 打开/关闭主面板
- `/ctk help` - 显示帮助信息
- `/ctk clear` - 清除数据
- `/ctk team on/off` - 开启/关闭团队通知

## 功能

- **自动检测**：自动检测地图图标，更新刷新时间
- **位面追踪**：自动追踪位面ID变化（需鼠标悬停NPC获取）
- **数据保存**：所有数据自动保存，关闭游戏后不丢失
- **团队通知**：可开启团队通知功能，空投事件自动通知队友
- **多语言支持**：支持中文和英文，根据游戏语言自动切换

## 本地化配置

插件支持中英文双语，检测文本已本地化配置：

- **中文版**：已完整配置，可直接使用
- **英文版**：已完整配置，可直接使用
