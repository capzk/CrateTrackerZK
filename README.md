# 空投物资追踪器

魔兽世界空投箱子刷新追踪插件，自动记录和追踪空投箱子的刷新时间和位面信息。

## 快速开始

1. 输入刷新时间：点击表格中的"上次刷新"列，输入时间（格式：HH:MM:SS 或 HHMMSS）
2. 查看倒计时：主面板实时显示各地图的下次刷新倒计时
3. 获取位面 ID：将鼠标指向任意 NPC 即可获取当前位面 ID

## 命令

- `/ctk` - 打开/关闭主面板
- `/ctk help` - 显示帮助信息
- `/ctk clear` - 清除数据并重新初始化插件
- `/ctk team on/off` - 开启/关闭团队通知（无 status 查询）

## 功能

- **自动检测**：仅地图图标（Vignette）检测，自动更新刷新时间
- **位面追踪**：位面 ID 显示（需将鼠标指向任意 NPC 获取）
- **数据保存**：SavedVariables 持久化，重载/重登不丢失
- **团队通知**：可按需开启团队通知
- **多语言支持**：支持 zhCN / zhTW / enUS / ruRU，自动按客户端语言切换
