# 空投物资追踪器

魔兽世界空投箱子刷新追踪插件，帮助玩家记录和追踪游戏中各种空投箱子的刷新时间和位面信息。

## 版本

v1.0.7

## 使用方法

### 基本操作

1. **打开/关闭主面板**
   - 使用命令：`/ct` 或 `/crate`
   - 或点击游戏界面中的浮动按钮

2. **查看刷新时间**
   - 打开主面板，查看各个地图的下次刷新时间
   - 支持按地图名称、上次刷新、下次刷新等列排序

3. **更新刷新时间**
   - 点击地图行，输入上次刷新时间（格式：HH:MM:SS 或 HHMMSS）

4. **移动浮动按钮**
   - 拖动浮动按钮到合适位置，位置会自动保存

## 命令列表

### 基本命令

- `/ct` 或 `/crate` - 打开/关闭插件主面板
- `/ct help` - 显示帮助信息

### 数据管理

- `/ct clear` 或 `/ct reset` - 清除所有时间和位面数据（保留地图列表）

### 团队通知

- `/ct team on` - 开启团队通知
- `/ct team off` - 关闭团队通知
- `/ct team status` - 查看团队通知状态

## 功能说明

### 自动检测

- **空投事件检测**：自动检测NPC喊话和地图图标，更新刷新时间
- **位面ID追踪**：自动追踪当前地图位面ID变化
- **数据持久化**：所有数据自动保存，关闭游戏后不会丢失

### 位面ID获取

- 无法从玩家自身获取位面信息，必须通过鼠标悬停或点击NPC才能获取到位面ID
- 进入地图时，会显示"当前位面ID为：X"
- 位面ID变化时，会显示"地图[地图名称]位面已变更：Y"

### 通知功能

- **玩家通知**：始终通知玩家（聊天窗口）
- **团队通知**：可通过命令开启/关闭（默认关闭）
  - 开启后，空投事件通知会同时发送到团队聊天（小队/团队/副本聊天）
